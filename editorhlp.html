Руководство пользователя

Настоящее руководство описывает функциональные возможности программного комплекса ГеоРедактор, предназначенного для импорта, векторизации, редактирования, пространственного анализа и верификации геометрии кадастровых и пространственных данных.

⌨ Горячие клавиши

Программа поддерживает управление с клавиатуры для ускорения выполнения основных операций картографирования и редактирования контуров:

Клавиша / Сочетание Назначение операции
F1 Вызов справочного руководства пользователя в новой вкладке браузера.
Home Масштабирование и автоматическое центрирование вьюпорта на выделенном объекте. Если выделение отсутствует — масштабирование по границам всех видимых пространственных данных.
End Принудительное позиционирование камеры и установка фиксированного масштаба (800) в точке привязки, расположенной непосредственно под курсором мыши.
F6 Копирование в буфер обмена координат текущего положения курсора или параметров подсвеченной вершины (при наведении) в стандартном формате X[табуляция]Y.
Esc Прерывание активных процедур (измерение, интерактивное рисование, деактивация режима изменения геометрии) или отмена незавершенных сетевых запросов загрузки геометрии из внешних СУБД/API.
+ / = Увеличение масштаба отображения рабочей области относительно геометрического центра экрана.
- / _ Уменьшение масштаба отображения рабочей области относительно геометрического центра экрана.

🖱 Навигация и интерфейс

Методы управления рабочей областью:
• Перемещение (панорамирование): Зажмите левую кнопку мыши (ЛКМ) на свободном участке холста и перемещайте указатель. В режиме редактирования геометрии сдвиг карты осуществляется перетаскиванием областей вне узловых интерактивных точек.
• Масштабирование: Вращение колесика мыши. Фокус масштабирования автоматически удерживается на текущем экранном положении курсора.
• Выделение объекта: Одиночный клик ЛКМ по телу полигонального объекта или по линии.
• Быстрые действия с колесиком мыши (СКМ): Одиночный клик СКМ при высоком масштабе сбрасывает вьюпорт на общий обзор всех объектов. При отдалении клик СКМ мгновенно центрирует и масштабирует область к ближайшей вершине или сегменту по алгоритму привязки. Двойной клик СКМ с интервалом менее 300 мс сбрасывает отображение на глобальные границы данных.

Вкладки боковой панели

Интерфейс управления параметрами разделен на три функциональные вкладки:

  1. Свойства: Отображает метрические параметры выделенного объекта (ID, тип, периметр, площадь, количество контуров и вершин), позволяет настраивать его цвет, открывает доступ к локальным операциям и списку пространственных наложений.
  2. Координаты: Содержит интерактивную таблицу координат вершин выделенного объекта.
  3. Объекты: Представляет собой дерево слоев проекта. Поддерживает операцию Drag-and-Drop для физического изменения порядка отрисовки объектов на холсте, а также групповые операции управления видимостью и удалением данных.

📥 Импорт, экспорт и проекты

1. Универсальный импорт текстовых данных

Кнопка инициализирует Менеджер данных. Ввод в текстовое поле поддерживает следующие структуры данных:

  • Координатные пары: Ввод строк формата X Y или X[табуляция]Y. Пустая строка в массиве интерпретируется как разделитель контуров (переход от внешнего контура к внутренним вырезам/дыркам).
  • Кадастровые номера: Списки номеров земельных участков, зданий или сооружений в формате XX:XX:XXXXXXX:XX (каждый номер с новой строки) для пакетного импорта геометрии с геопортала ЕГРН/НСПД.
  • Кадастровый квартал: Номер квартала вида XX:XX:XXXXXX или XX:XX:XXXXXXX. Запускает диалоговое окно выбора фильтра объектов для извлечения данных из соответствующего XML-архива КПТ или через API НСПД. Загрузка нулевых кварталов заблокирована.
  • Именованные контуры: Первая строка — произвольное наименование объекта (или его кадастровый номер), со второй строки — координаты вершин. Допускается импорт нескольких объектов, разделенных строками с наименованиями.
  • Функция «Найти»: Кнопка считывает координаты из поля ввода и строит временные точечные метки-прицелы (класс crosshair) с подписями их значений, не искажая структуру существующих слоев.

2. Перепроецирование и режимы загрузки ЕГРН

Выпадающий список режимов загрузки кадастровых номеров определяет метод разбора:

  • Из архива КПТ (XML файл): Запрос и парсинг XML-документов кадастровых планов территорий непосредственно из локальной базы данных или удаленного хранилища.
  • Системы координат (МСК -> EPSG): Активирует автоматическое проецирование пространственных данных из глобальной системы координат WGS84 / Web Mercator (EPSG:3857, используемой публичными картами) в местную систему координат (МСК) выбранного региона России на основе встроенной библиотеки проекций proj4.js.
  • Фиксированный цвет: Чекбокс принудительного присвоения единого цветового тона всем объектам загружаемого пакета.

3. Импорт из файлов

Функция загрузки файлов поддерживает пакетный разбор, чтение ZIP-архивов и обработку следующих форматов:

  • .txt, .csv — текстовые структурированные ведомости координат. Поддерживается разбор формата CSV Технокад (распознавание идентификаторов контуров в квадратных скобках [1]).
  • .mif / .mid — векторный формат MapInfo. Корректно разбирает геометрию типов Region, Pline, Line и Point.
  • .dxf — формат обмена чертежами AutoCAD. Поддерживает примитивы POLYLINE, LWPOLYLINE, LINE, POINT, CIRCLE. Программа считывает текстовые аннотации (TEXT, MTEXT) и предлагает ассоциировать их с ближайшими объектами в качестве наименований.
  • .geojson / .json — стандартные географические данные JSON (типы геометрии Polygon, MultiPolygon, LineString).
  • .kml — формат пространственных данных Keyhole Markup Language.
  • .xyh — файлы геодезической съемки (высотные отметки игнорируются).
  • .sxf — бинарные файлы цифровых карт.
Фильтрация выбросов при импорте:
Для форматов DXF, SXF и общих текстовых списков применяется алгоритм фильтрации грубых ошибок измерений. Программа вычисляет медианный центр импортируемого облака точек и автоматически удаляет вершины, удаленные от медианы более чем на 5 километров, предотвращая искажение масштаба холста из-за единичных ошибочных координат в нулевой точке.

4. Экспорт

  • Экспорт MIF: Полноценный экспорт всех видимых объектов схемы. Генерирует файл геометрии .mif (в прямоугольной системе координат NonEarth) и файл семантики .mid, содержащий структурированную таблицу из 39 стандартных атрибутивных полей кадастрового учета. Данные упаковываются в ZIP-архив.
  • Сохранение проекта (.geoed): Экспорт полной рабочей сессии в JSON-файл проекта. Сохраняет структуру объектов, цвета, метаданные, параметры вьюпорта и состояние растровой подложки.

🖼 Растровая подложка

Редактор поддерживает работу с геопривязанными растровыми изображениями, используемыми в качестве картографической основы:

  • Загрузка: Растровые подложки импортируются в проект автоматически при открытии комплексных файлов проекта .geoed или специализированных файлов импорта GeoredactorWithRaster.
  • Управление видимостью: Кнопка на холсте включает или отключает отображение растрового слоя на Canvas под векторными объектами.
  • Позиционирование: Положение растра на экране жестко привязано к координатной сетке МСК на основе параметров привязки угловых точек (boundsMsk: { topLeft, bottomRight }). Границы растра учитываются в алгоритмах автоматического расчета общего обзора проекта при нажатии клавиши Home.

📐 Рисование и измерения

1. Интерактивное рисование контуров

Режим запускается кнопкой . На панели «Создание объекта» доступен выбор типа контура:

  • Свободный контур:
    • Клик ЛКМ устанавливает вершину. При приближении к существующим точкам срабатывает магнитное притягивание (радиус захвата — 10 пикселей).
    • Клик ПКМ отменяет установку последней созданной вершины.
    • Двойной клик ЛКМ завершает построение, замыкает фигуру и преобразует её в тип polygon.
    • Нажатие кнопки завершения фиксирует контур без замыкания в тип line (или point при наличии одной вершины).
  • Прямоугольник (2 точки):
    • Позволяет строить прямоугольные объекты с фиксированными параметрами длины и ширины в метрах, заданными в полях ввода на панели.
    • Первый клик ЛКМ фиксирует опорную точку (якорь).
    • Перемещение мыши задает направление (вектор поворота) прямоугольника относительно горизонтальной оси. Геометрия рассчитывается интерактивно в реальном времени.
    • Второй клик ЛКМ фиксирует угол поворота и строит замкнутый полигональный контур.

2. Инструмент «Линейка»

Активируется кнопкой . Позволяет измерять ортодромические расстояния по цепочке точек. На экране отображаются длина каждого сегмента, общая длина полилинии и расстояние по прямой между первой и последней вершиной (для незамкнутых трасс из более чем двух узлов).

✏ Редактирование объектов

Режим редактирования геометрии выбранного объекта активируется кнопкой . Включение режима сопровождается статической красной рамкой по периметру экрана.

1. Изменение положения вершин

Перетаскивание узла осуществляется зажатием ЛКМ на маркере вершины. При перемещении рассчитываются новые координаты. Дополнительно включаются следующие механизмы обратной связи:

  • Динамический след («Trail»): На экране отображается фиолетовая пунктирная линия, соединяющая исходное положение вершины с её новым положением во время перетаскивания. Над линией выводится расстояние смещения в метрах. След сохраняется на экране в течение 5 секунд после отпускания кнопки мыши для контроля величины сдвига.
  • Интерактивное изменение площади: Для полигонов во вкладке «Свойства» в процессе движения точки динамически отображается строка вида: [Старая площадь] → [Новая площадь] (Разница).

2. Добавление и удаление вершин

  • Добавление: При наведении курсора на сегмент границы редактируемого объекта указатель принимает вид перекрестия. Клик ЛКМ внедряет новую вершину в контур в точке клика.
  • Удаление: Клик ПКМ по маркеру вершины удаляет её из геометрии после подтверждения в диалоговом окне. Удаление блокируется при достижении минимального лимита вершин (3 для полигона, 2 для линии).

3. Инструмент «Свободный перенос»

Кнопка активирует параллельный перенос всего объекта. Зажатие ЛКМ в любой точке объекта позволяет перемещать его по карте без изменения внутренней геометрии контуров и взаимного положения вырезов.

🛠 Инструменты геометрии

Панель дополнительных инструментов становится активной при выделении объекта в режиме редактирования геометрии:

1. Смещение границ (Буферная зона)

Инструмент строит эквидистантные смещения контуров:

  • Полигоны: положительные значения строят смещение наружу контура, отрицательные — внутрь (сжатие).
  • Линии: строит замкнутый полигональный буфер вокруг осевой линии на заданном расстоянии во все стороны скруглением торцов.

2. Рассечение объекта (Ножницы)

Инструмент выполняет пространственное деление контуров:

  • Полигон-Полигон: вычитает геометрию полигона-резака из целевого полигона.
  • Полигон-Линия: производит точное рассечение полигона линией без создания буферной зоны, разделяя исходный объект на два или более самостоятельных полигона с распределением внутренних вырезов (дырок).
  • Линия-Линия (Резак): разрезает линейный объект в точках его физического пересечения с контуром резака.

3. Объединение объектов

Инструмент объединяет два полигона в единый пространственный контур. Пересекающиеся внешние границы сливаются, внутренние изолированные пустоты сохраняются.

4. Интерактивные точки пересечения

Кнопка вычисляет координаты точек пересечения ребер двух контуров и предлагает четыре режима обработки:

  1. Создать только точки: генерирует новые независимые точечные объекты на карте.
  2. Создать точки в обоих полигонах: физически внедряет новые вершины в состав геометрии обоих пересекающихся полигонов.
  3. Создать точки только в объекте №1: модифицирует геометрию только первого выбранного полигона.
  4. Создать точки только в объекте №2: модифицирует геометрию только второго выбранного полигона.

Кнопка «Создать точки пересечения текущего со всеми» выполняет аналогичный расчет выбранного полигона со всеми остальными видимыми полигонами проекта в один клик.

5. Создание геометрических фигур по точкам

Инструмент выполняет пакетную генерацию фигур, центрированных на вершинах редактируемого объекта:

  • Квадрат: Строит квадраты с заданной длиной стороны и углом поворота в градусах.
  • Круг: Строит аппроксимированные круги с заданным радиусом и шагом точек (длиной дуги сегментации).
  • Многоконтурность: Опция объединения всех сгенерированных фигур в один многоконтурный объект.

6. Добавление ветки к линии

Кнопка позволяет рисовать дополнительные изолированные контуры для линейных объектов (создание составных мультилиний).

📋 Таблица и её меню

Таблица координат во второй вкладке панели представляет список вершин в порядке обхода контуров.

1. Интерактивные функции

  • Одиночный клик по строке таблицы подсвечивает вершину на холсте статичным красным маркером-мишенью.
  • Двойной клик по значению ячейки (X или Y) копирует текстовое значение координаты в буфер обмена.
  • Кнопка копирования в заголовке контура таблицы копирует весь список координат данного контура в текстовом формате.

2. Контекстное меню операций над контуром

Клик правой кнопкой мыши по любой значащей строке таблицы координат открывает контекстное меню:

  • Обратный порядок точек: Меняет порядок обхода контура на противоположный (реверс направления). Для полигонов автоматически пересчитывается корректное положение замыкающей вершины (первая копируется в конец).
  • Сделать точку первой: Назначает выбранную вершину стартовой точкой контура, циклически сдвигая массив координат без изменения геометрической формы объекта.
  • Авто (СЗ, по часовой): Принудительно устанавливает направление обхода по часовой стрелке, а первой точкой назначает вершину, наиболее близкую к идеальному северо-западному углу описывающего прямоугольника.
  • Авто для всех полигонов: Выполняет операцию нормализации (Северо-Запад, по часовой стрелке) для всех полигональных объектов проекта в пакетном режиме.
  • Поменять X Y местами: Выполняет глобальное транспонирование осей координат (обмен значений X и Y) для абсолютно всех объектов в проекте, автоматически пересчитывая масштабы, центроиды и границы рабочей области холста.

👁 Анализ и визуализация

1. Режимы отображения вершин

Для оптимизации визуального восприятия и производительности рендеринга предусмотрены режимы фильтрации маркеров точек:

  • Глобальный фильтр: Кнопка включает или выключает отображение маркеров всех вершин. При импорте или создании суммарно более 2000 вершин на холсте маркеры точек автоматически отключаются для предотвращения снижения производительности рендеринга.
  • Показать точки смежных (1м): Кнопка оставляет видимыми вершины только для выделенного объекта и объектов, ребра или узлы которых удалены от него на расстояние менее 1 метра.
  • Показать точки текущего объекта: Кнопка скрывает вершины всех слоев, кроме выделенного.

2. Нумерация и метки

  • Сквозная нумерация точек: Кнопка включает непрерывный возрастающий порядок номеров вершин по всем видимым на холсте объектам (в порядке их расположения в списке слоев). При отключении нумерация ведется независимо для каждого контура начиная с единицы.
  • Подписи координат: Кнопка выводит на карту выносные полки с точными значениями координат X и Y для каждой вершины. Алгоритм рассчитывает углы наклона выносок в четырех квадрантах и предотвращает наложения текста на основе детекции коллизий экранных прямоугольников.
  • Размеры сторон: Кнопка переключает отображение длин сегментов в метрах в трех режимах: 'off' (отключено), 'selected' (только для выделенного объекта), 'all' (для всех сегментов всех видимых объектов схемы).
  • Текстовые метки названий: Названия слоев отрисовываются на карте по вычисленному центроиду геометрии. В диалоговом окне переименования доступна настройка индивидуального размера шрифта названия в диапазоне от 8 до 72 пикселей.

3. Инструменты пространственного анализа отношений

  • Изоляция смежных в радиусе 20м: Кнопка скрывает все объекты схемы, за исключением выделенного и контуров, находящихся в радиусе 20 метров от него (включая пересекающиеся, вложенные и смежные объекты).
  • Анализ наложений: В левой панели выводится список наложений, пересечений и касаний выделенного полигона с другими фигурами. Клик по строке списка центрирует камеру на объекте-партнере.
  • Изоляция наложений: Кнопка «Показать наложения» в свойствах объекта скрывает все слои карты, не имеющие пересечений площади с выделенным объектом.
  • Переход по точкам нарушений: Кнопка на панели наложений последовательно перемещает камеру и центрирует вьюпорт на вершинах выделенного объекта, которые физически расположены внутри границ других полигонов схемы.
  • Контроль близких точек: Редактор автоматически тестирует геометрию выделенного контура. Вершины, расстояние между которыми составляет менее 10 сантиметров, помечаются на карте предупреждающими оранжевыми концентрическими окружностями.

4. Внешняя визуализация данных

  • Показать на Схеме: Кнопка копирует координаты всех видимых объектов в буфер обмена и открывает их во внешнем модуле визуализации схем (sh.html) в новой вкладке.
  • Открыть ЧЗУ: Кнопка копирует координаты с разделением по заголовкам объектов и открывает их во внешнем чертежном модуле частей земельных участков (sch.html).
