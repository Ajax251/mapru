Сведения об объекте недвижимости

Модуль «Сведения ЕГРН» предназначен для получения семантической (текстовой) информации об объектах недвижимости на основе данных Росреестра и Национальной системы пространственных данных (НСПД).

Модуль выводит характеристики объектов, сведения о правах и кадастровой стоимости, формируя структурированную карточку для просмотра, копирования данных или вывода на печать.

Способы поиска и загрузки данных

В модуле реализовано несколько способов ввода кадастрового или реестрового номера для запроса информации.

Способ Описание механизма
Чтение буфера обмена При открытии страницы выполняется проверка буфера обмена операционной системы. Если скопированный текст соответствует формату кадастрового номера, загрузка данных по этому объекту начинается автоматически.
Ручной ввод Допускается ввод числовых значений без разделителей в поисковую строку. При вводе применяется автоматическое форматирование к стандарту ХХ:ХХ:ХХХХХХХ:ХХ.
Переход с карты При вызове функции «ЕГРН» из контекстного меню основной карты, идентификатор выбранного объекта (ЗУ, ОКС или ЗОУИТ) передается в данный модуль для формирования карточки.

Адаптация под типы объектов

Структура отображаемых данных и набор полей карточки зависят от типа запрашиваемого объекта недвижимости. Пустые поля скрываются автоматически.

Тип объекта Отображаемые параметры
Земельные участки (ЗУ) Выводятся сведения о категории земель, видах разрешенного использования по документу и типе площади. Указывается принадлежность к кадастровому кварталу.
Здания и помещения (ОКС) Отображаются архитектурные характеристики: материал наружных стен, этажность (включая подземные этажи), год завершения строительства или ввода в эксплуатацию, а также назначение.
Сооружения (Линейные) В качестве основных параметров выступают протяженность (в метрах), глубина или объем, а также наименование сооружения согласно технической документации.
ЗОУИТ и Территории Указывается реестровый номер, тип установленной зоны и текстовое описание ограничений и обременений использования территории.

Структура данных карточки

Раздел Состав данных
Общая информация Кадастровый номер, статус записи (Учтенный / Ранее учтенный / Снят с учета), даты присвоения статуса и обновления информации.
Характеристики Площадь, протяженность, этажность. Указывается статус площади земельного участка (уточненная или декларированная).
Кадастровая стоимость Полная стоимость, удельный показатель кадастровой стоимости (УПКС), дата утверждения оценки и дата внесения в базу данных.
Местоположение Адрес объекта. Для участков дополнительно выводятся данные о ВРИ и категории земель.
Права и ограничения Сведения о форме собственности и видах зарегистрированных прав (собственность, аренда, постоянное бессрочное пользование).

Связи между объектами

При наличии соответствующих записей в базах данных (ЕГРН/НСПД), модуль отображает перекрестные ссылки между связанными объектами недвижимости.

Связанные ОКС (от ЗУ) При просмотре информации о земельном участке выполняется запрос связанных ОКС. При их наличии формируется список в виде гиперссылок для перехода к карточкам соответствующих зданий.
Родительский ЗУ (от ОКС) При просмотре карточки здания или сооружения отображается кадастровый номер земельного участка, в границах которого расположен данный объект (в виде ссылки).
Кадастровый квартал Номер кадастрового квартала объекта является интерактивным элементом. Переход по ссылке инициирует загрузку информации о квартале на главной карте.

Панель инструментов

Инструмент Описание функции
Показать на карте Осуществляет переход на вкладку основной карты с автоматическим позиционированием, отрисовкой и подсветкой контура текущего объекта по его координатам.
Печать Копирует в буфер обмена кадастровый номер и открывает программу "Печать сведений"
Копировать текст Формирует текстовый массив из заполненных полей карточки (в формате «Ключ: Значение») и копирует его в буфер обмена операционной системы для последующей вставки в документы.

Горячие клавиши Функции

Перечень поддерживаемых сочетаний клавиш для навигации и управления функциями модуля.

Клавиша Описание действия
F1 Открытие документации (данного руководства пользователя).
F8 Вызов режима печати карточки объекта.
F9 Переключение видимости экранной цифровой панели (Numpad) для ввода данных с сенсорных устройств.
Enter Инициализация поиска по номеру, введенному в поисковую строку.
Ctrl + C При наличии выделенного фрагмента копируется выделенный текст.
Ctrl + P Вызов диалогового окна печати.
Ctrl + V Вставка содержимого буфера обмена в строку поиска (при наличии фокуса в поле ввода).
Esc Очистка строки поиска и снятие фокуса с активных элементов управления.
