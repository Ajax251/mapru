Установка и настройка Windows PATH
Чтобы BAT-файл находил команду gswin64c (без указания полного пути), необходимо, чтобы папка bin установленного Ghostscript была добавлена в переменные среды Windows (PATH).
Настройка путей к программе
Добавить путь вручную в переменные среды
rundll32 sysdm.cpl,EditEnvironmentVariables
Добавить путь через PowerShell (замените путь на свой, если он отличается)
$addPath = "C:\Program Files\gs\gs10.04.0\bin"; $currentPath = [Environment]::GetEnvironmentVariable("Path", "User"); if ($currentPath -notmatch [regex]::Escape($addPath)) { [Environment]::SetEnvironmentVariable("Path", $currentPath + ";" + $addPath, "User"); Write-Host "Добавлено!" }
