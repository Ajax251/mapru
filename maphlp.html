Руководство пользователя: Карта

Данная программа представляет собой инструмент для работы с кадастровой картой, пространственными данными и координатами. Ниже представлено детальное описание функционала, доступного через интерфейс управления и контекстное меню.

Панель управления (Виджет)

Основная панель в левом верхнем углу экрана предоставляет доступ к базовым функциям навигации и настройки.

Элемент Функция
Режимы карты Переключение подложек: Схема, Спутник, Гибрид, Пустая карта (для чистого экспорта графики).
Найти по координатам Запускает поиск и конвертацию списка координат, введенных в текстовое поле (F9).
Конвертер МСК Открывает инструмент для пересчета координат из Местных Систем Координат в общемировую систему WGS84.
Настройки Управление визуальным стилем (толщина линий, цвета заливки, цвета контуров) и параметрами смещения карты.
Текст Добавление произвольных текстовых заметок на карту с возможностью их перемещения.
Внешние карты Быстрый переход к текущей локации в сторонних картографических сервисах (Яндекс, Google, 2GIS).
Поиск Поле для ввода адреса, кадастрового номера или координат для быстрого перехода.

Контекстное меню

Основной функционал программы доступен через нажатие правой кнопки мыши на карте.

Раздел "Квартал"

Пункт меню Действие
Земельные участки в квартале Загружает границы земельных участков, относящихся к выбранному кадастровому кварталу.
Здания в квартале Отображает здания (ОКС) в данном квартале.
Сооружения в квартале Загружает линейные и площадные сооружения в пределах квартала.
ЗОУИТ в квартале Показывает Зоны с Особыми Условиями Использования Территории для всего квартала.
Границы квартала Отрисовывает границы кадастрового квартала.
Собственность (Тема) Включает тематическую раскраску участков по типу собственности.
Разрешенное использование (Тема) Раскрашивает участки в зависимости от их ВРИ.
Стоимость (Тема) Отображает тепловую карту кадастровой стоимости участков.
Дата внесения (Тема) Визуализация участков по дате их регистрации в реестре.
Единое землепользование (Тема) Подсвечивает участки, входящие в состав единого землепользования.
Сбросить Отключает все тематические раскраски и фильтры.
Архив КПТ Попытка загрузки данных из сохраненного локального архива квартала.
Обновить ЕЗ Запускает процедуру актуализации данных единого землепользования.
Экспорт Выгрузка табличных данных и геометрии текущего квартала в формат ZIP/Excel.

Раздел "Объект"

Пункт меню Действие
ЕГРН Открывает сведения об объекте
Открыть КН в XML Генерирует XML-схему для выбранного кадастрового номера.
Связанные ОКС на ЗУ Выполняет поиск зданий, которые юридически привязаны к выбранному земельному участку.
ОКС на объекте Геометрический поиск всех ОКС, которые накладываются на контур выбранного полигона.
Сооружения на объекте Геометрический поиск всех сооружений, пересекающих выбранный полигон.
ЗОУИТ на объекте Поиск охранных зон и ограничений, накладывающихся на выбранный объект.
Отчет (площадной) Формирование сводного отчета с анализом пересечений для площадного объекта (ЗУ/Здание).
Отчет (линейный) Формирование отчета для линейного объекта с каталогом координат и анализом пересечений.
Экспорт в KML Google Earth Сохранение геометрии выбранного объекта в файл .kml для просмотра в Google Earth.

Раздел "Границы"

Пункт меню Действие
Открыть ЗУ Открывает земельный участок в точке клика.
Открыть ОКС Открывает объект капитального строительства в точке клика.
Территориальные зоны Отображает границы территориальных зон в указанной точке.
ЗОУИТ Показывает границы зон с особыми условиями использования территории в точке клика.
Населенные пункты Отображает границы населенных пунктов.
Муниципальные образования Показывает границы муниципальных районов или округов.
Лесничество Показывает границы лесничеств и лесопарков.
Импорт из JSON Позволяет вставить сырые данные (GeoJSON/JSON) для отображения произвольной геометрии.

Раздел "МСК"

Пункт меню Действие
Объект в МСК Конвертирует координаты выбранного полигона в установленную МСК и копирует в буфер.
Все объекты в МСК Массовая конвертация всех объектов на карте в МСК с копированием в буфер обмена.
Георедактор Экспорт координат объектов в формат, совместимый с внешним георедактором.
ЗУ в XML КПТ Формирование XML-файла кадастрового плана территории для загруженных участков.
Все объекты в XML Генерация единого XML-файла схемы, включающего все отрисованные объекты.

Инструменты карты

Пункт меню Действие
Расстояние Инструмент линейки. Клик — новая точка, двойной клик — завершение измерения.
Площадь Инструмент измерения площади и периметра произвольного полигона.
Местоположение Определяет текущее местоположение устройства и центрирует карту.
Наложения Инструмент проверки пересечений загруженных объектов с заданным контуром.
Копировать Копирует кадастровый номер или название объекта под курсором.
Координаты > Точка в WGS84 Копирует координаты точки клика в формате Широта, Долгота.
Координаты > Карта Мини Копирует координаты и открывает облегченную версию карты.
Координаты > Все в EPSG:3857 Экспорт координат всех объектов в проекции Меркатора.
Экран Создает скриншот текущей области карты (без интерфейса) и копирует в буфер.
Растр Открывает панель управления растровыми подложками (загрузка, привязка, прозрачность).
Изменить Открывает редактор стиля выбранного полигона (цвет, прозрачность, толщина линий).

Горячие клавиши

Клавиша Действие
Enter Запуск поиска / Подтверждение ввода.
F1 Руководство пользователя.
F7 Быстрый экспорт данных в архив.
F8 Запуск импорта данных из облачного хранилища.
F9 Показать/скрыть панель ввода координат.
F10 Сделать скриншот карты.
Delete Удаление выделенного объекта (линии, метки).
Esc Отмена текущего действия (рисования, измерения), закрытие окон.
+ / - Приближение / Отдаление карты.
